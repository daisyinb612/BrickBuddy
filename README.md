# BrickBuddy

This repository hosts the public project page for BrickBuddy, a step-situated
wearable agent for knowledge exploration during brick assembly.

Demo video: https://www.youtube.com/watch?v=3-L7GyE6FCs

The project page is in `docs/` so it can be served with GitHub Pages.

Source code and setup notes are available in `BrickBuddy_github/`.
